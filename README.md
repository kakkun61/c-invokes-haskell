# The sample of the FFI from C (GCC) to Haskell (GHC)

## Build

```
> make
```

## Run

```
> c-invokes-haskell.exe
```

## Used versions

```
> ghc --version
The Glorious Glasgow Haskell Compilation System, version 8.10.1
```
